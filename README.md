# Personal Portfolio Website

## Project Overview

This is a responsive personal portfolio website built using HTML, CSS, and Bootstrap 5. The website is designed to showcase my skills, projects, and contact information in a clean, modern, and professional manner.


## Features

* Fully responsive design (Mobile, Tablet, and Desktop)
* Fixed navigation bar with smooth scrolling
* Home section with personal introduction
* Skills section highlighting technical abilities
* Projects section showcasing completed work
* Contact section for communication
* Modern user interface using Bootstrap 5
* Interactive project cards with hover effects
* Clean and user-friendly layout


## Technologies Used

* HTML5
* CSS3
* Bootstrap 5

## How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

### Using VS Code

1. Install the Live Server extension.
2. Right-click on `index.html`.
3. Select **Open with Live Server**.
